# Shadow Architect Landing Page

A sophisticated, modern landing page for Shadow Architect Consulting - a boutique consultancy firm specializing in maximum influence psychology frameworks.

## Features

- **Modern Next.js 14** with TypeScript
- **Tailwind CSS** for elegant, dark-themed styling
- **Responsive Design** optimized for all devices
- **Sophisticated UI** matching the Shadow Architect Framework tone
- **Seven Dominion Pillars** showcase
- **Contact Form** for consultation requests
- **Philosophy Section** explaining the framework's core principles

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```

## Design Philosophy

The landing page reflects the Shadow Architect Framework's sophisticated, authoritative tone:

- **Dark Theme**: Deep blacks and gold accents representing power and exclusivity
- **Serif Typography**: Classic, authoritative font choices
- **Minimalist Layout**: Clean, focused presentation
- **Powerful Messaging**: Direct, confident language matching the framework
